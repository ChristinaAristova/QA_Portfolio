# Этап 2: Функциональное тестирование UI/UX

В рамках второго этапа обучения на базе сайта **Hitch** (сервис для поиска аутстаф-специалистов) были изучены и применены ключевые аспекты функционального тестирования.

### Ключевые навыки:

-   **Методологии тестирования:** Применение исследовательского и интуитивного тестирования для оперативной оценки продукта.
-   **Техники тест-дизайна:** Использование классов эквивалентности и анализа граничных значений для оптимизации тестового покрытия.
-   **Тестирование UI/UX:** Проведение комплексного тестирования основных элементов интерфейса, форм и пользовательских сценариев.
-   **Работа с требованиями:** Навыки тестирования и анализа требований, а также составление базовой тестовой документации (тест-планы, отчеты о тестировании).
-   ~~**Предугадывание ошибок (Error Guessing):**~~* Применение пользовательского опыта для целенаправленного поиска слабых мест.
-   ~~**Попарное тестирование:**~~* Сокращение числа тестов за счет комбинации параметров парами.

### Инструменты:

Активное использование системы баг-трекинга **YouGile** для создания и управления баг-репортами.

### Дополнительные аспекты (в процессе изучения):
